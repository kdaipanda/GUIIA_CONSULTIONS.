# Pitch 3 minutos (con discovery corto)

## 1. Contexto (20 s)

> Trabajo con GUIAA, software clínico para veterinarios certificados. No vengo a venderles una membresía a ciegas: primero quiero entender cómo registran hoy las consultas.

## 2. Discovery (60–90 s)

Elegir 2–3 de `discovery-questions.md`. Escuchar. Repetir el dolor en sus palabras.

## 3. Encaje (60 s)

Mapear su dolor → GUIAA:

| Si dijeron… | Tú conectas con… |
|-------------|------------------|
| Papel / Excel | Historial digital por paciente, searchable |
| WhatsApp para casos | Consulta estructurada + soporte CDS |
| Muchas especies | Flujo multiespecie (no solo perro/gato) |
| “No hay tiempo” | 3 consultas gratis; valor se ve en la 1ª demo |
| Ya tienen software | Qué les frustra → contrastar claridad clínica |

## 4. Oferta + prueba (30 s)

> Arrancan con **3 consultas de prueba** sin tarjeta. Si les sirve, hay planes Basic / Professional / Premium. Cupón de campo cuando aplique: [promo vigente].

## 5. CTA (20 s)

> ¿Registramos ahora en 60 segundos o le dejo el QR y le mando el link por WhatsApp?

---

**No digas:** “IA que diagnostica sola”, “reemplaza al veterinario”, promesas de cura.
**Sí di:** apoyo a la decisión clínica, historial, productividad en consulta.
